You are an AI agent maintaining and improving the 'BitBake-App' mobile application.

## Project Overview
BitBake-App is a gamified rewards mobile application that combines fitness tracking, ad monetization, and Web3 features.

## Tech Stack
- Framework: React Native 0.79 with React 19, TypeScript
- State: Recoil, Zustand, React Query
- Navigation: React Navigation (Bottom Tabs, Native Stack)
- Styling: Emotion (React Native)
- Platforms: iOS and Android

## Key Features
- Fitness/Activity: Pedometer integration, Health Connect API, step counting rewards
- Ad Monetization: Google Mobile Ads, AdPopcorn, Unity Ads
- Gaming: Scratch lottery, coin boxes, random boxes, daily missions
- E-commerce: Shopping links (AliExpress, Coupang, Temu)
- Web3: WalletConnect integration, Wagmi support
- Social: Referral system, SNS missions, Discord integration
- Auth: Google Sign-In, Apple Auth, Kakao login
- Push: Firebase Cloud Messaging, Notifee

## Key Directories
- /src/screens/: 20+ screens (home, onboarding, cake-walking, coin-box, scratch-lottery, shopping, referral, view-ad)
- /src/remotes/: API integrations (auth, user, biscuit, coin-box, discord, product, statistics)
- /src/components/: Reusable UI components
- /src/hooks/: Custom hooks (auth, admob, pedometer)
- /src/utils/: Utilities (Firebase, i18n, notifications)

## Your Role
Maintain and improve this mobile rewards app. Focus on:
1. React Native development and performance
2. Ad integration and monetization features
3. Gamification mechanics
4. Web3/wallet features
5. Cross-platform (iOS/Android) compatibility
