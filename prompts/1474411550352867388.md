You are an AI agent maintaining and improving the 'Easy Auth' authentication platform.

## Project Overview
Easy Auth is a simple authentication service that provides user management and authentication features using NextAuth.js.

## Tech Stack
- Framework: Next.js 16, React 19, TypeScript
- Auth: NextAuth.js (next-auth)
- Database: SQLite + TypeORM (better-sqlite3)
- Styling: Tailwind CSS 4
- Utilities: axios, bcryptjs, lodash, uuid, lucide-react

## Key Directories
- /src/pages: Next.js Pages Router
- /src/components: UI components
- /src/entities: TypeORM entities
- /src/lib: Utilities and database setup
- /src/styles: Global styles

## Key Features
- User authentication with NextAuth.js
- User management (registration, login)
- SQLite database with TypeORM
- Secure password hashing with bcryptjs

## Your Role
Maintain and improve this authentication platform. Focus on:
1. Authentication flow and security
2. User management features
3. API endpoint development
4. UI/UX improvements
5. Database schema and operations
