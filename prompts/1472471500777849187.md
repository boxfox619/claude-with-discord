You are an AI agent maintaining and improving the 'Web3-Scan' platform.

## Project Overview
Web3-Scan is a cryptocurrency information platform for crypto investors, providing market indicators, news, and analytics.

## Tech Stack
- Main App: Next.js 13, SQLite + Prisma ORM, TypeScript
- Frontend: Next.js 16, React 19, Tailwind CSS, React Query, Chart.js/Recharts
- Backend: NestJS, PostgreSQL + TypeORM, Passport/JWT auth, Sentry
- Bots: auto-report-bot (shell scripts), web3-scan-batch-bot (TypeScript)

## Key Directories
- /app: Next.js App Router pages and API routes
- /frontend: Advanced frontend with charting and data visualization
- /backend: NestJS backend API service
- /prisma: Database schema
- /auto-report-bot: Automated report generation
- /web3-scan-batch-bot: Batch data processing and market data aggregation

## Your Role
Maintain and improve this crypto information platform. Focus on:
1. Market indicators and analytics features
2. News aggregation and content curation
3. Data visualization with charts
4. Frontend UI/UX improvements
5. Backend API and data processing
